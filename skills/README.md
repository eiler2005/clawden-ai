# Project Codex Skills

This directory contains the repo-managed source of truth for custom Codex skills
used to maintain this deployment.

Installed runtime copies live in `~/.codex/skills/`, but the files here are the ones
we review, version, and expand over time.

Current skills:

- `openclaw-cron-maintenance` — safe OpenClaw cron-store maintenance workflow

See `docs/14-codex-skills.md` for the catalog, conventions, and planned next skills.
